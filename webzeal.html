Web Zeal

Rules for Web Zeal

  1. There will be three rounds in all

  2. Prelims :

    • 30 MCQ questions.

    • Questions will involve basic CSS, JS, AJAX, JSP, Servlets, PHP and other common web technologies.

  3. Pre-finals :

    • Three tasks of varying degrees of complexity will be provided.

    • Involves hands on application of javascript and proper use of CSS.

  4. Finals will involve full-fledged web application development

  5. Judgement based on :

    • Quality of website.

    • How it is rendered.

    • Multiple browser support.

    • Use of different technologies.

  6. Knowledge of PHP/JSP is a must.
